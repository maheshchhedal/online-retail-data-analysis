# 🛒 Online Retail Data Analysis

An end-to-end retail sales analysis project using **SQL**, **Python**, **Pandas**, and **Matplotlib** to answer real-world business questions. This project transforms raw retail transaction data into meaningful insights that can support business decision-making.

---

## 📌 Project Overview

The objective of this project is to analyze an online retail dataset and uncover insights related to:

- Customer purchasing behavior
- Sales and revenue trends
- Product performance
- Customer experience
- Shipping performance
- Payment methods

Rather than simply writing SQL queries, this project focuses on solving practical business problems commonly faced by retail and e-commerce companies.

# 🚀 How to Run

1. Clone the repository.

```bash
git clone https://github.com/maheshchhedal/online-retail-data-analysis.git
```

2. Install the required libraries.

```bash
pip install pandas matplotlib sqlalchemy pymysql jupyter
```

3. Open Jupyter Notebook.

```bash
jupyter notebook
```

4. Run all cells to reproduce the analysis.
